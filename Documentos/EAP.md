# 🌿 Artefato 7: Estrutura Analítica do Projeto (EAP)

Este documento divide o desenvolvimento do sistema de e-commerce de calçados em fases e entregas menores para facilitar o gerenciamento do projeto.

---

### 🗺️ Divisão das Entregas (Escopo do Projeto)

* **1. Gerenciamento e Documentação (Artefatos Fatec)**
  * 📄 1.1. Definição de Missão, Visão e Valores
  * 📋 1.2. Documento de Elicitação de Requisitos
  * 📊 1.3. Matrizes SWOT e 5W2H
  * 🔄 1.4. Desenho do Fluxo de Processo (BPMN)
  * ⚙️ 1.5. Documentação de Requisitos (RF e RNF)

* **2. Design da Interface (UI/UX)**
  * 🎨 2.1. Criação do protótipo de telas no Figma
  * 🚨 2.2. Definição da identidade visual dos Alertas Vermelhos (Clean UI)

* **3. Modelagem e Banco de Dados (Persistence)**
  * 📐 3.1. Desenho do Diagrama Entidade-Relacionamento (DER) no draw.io
  * 💾 3.2. Configuração do SQL Server Express
  * 🚀 3.3. Criação das Migrations com Entity Framework Core

* **4. Desenvolvimento do Software (C# / .NET)**
  * 📦 4.1. Implementação da camada `SuaLoja.Model` (Classes de Produto, Grade e Fornecedor)
  * ⚙️ 4.2. Implementação da camada de regras de negócio (Lógica de cruzamento de estoque)
  * 🖥️ 4.3. Construção das telas de administração e painel do estoque

* **5. Homologação e Entrega**
  * 🧪 5.1. Testes de validação com dados reais de fornecedores
  * 📦 5.2. Publicação do código final no GitHub
