# 🌿 Artefato 7: Estrutura Analítica do Projeto (EAP)

Este documento divide o desenvolvimento do nosso sistema em fases e tarefas diretas para facilitar a organização do projeto.

---

### 🗺️ Divisão das Tarefas

* **1. Documentos do Projeto**
  * 📄 1.1. Criar Missão, Visão e Valores
  * 📋 1.2. Fazer Elicitação de Requisitos
  * 📊 1.3. Montar as Matrizes SWOT e 5W2H
  * 🔄 1.4. Desenhar o fluxo de processos (BPMN)
  * ⚙️ 1.5. Listar os Requisitos do Sistema (RF e RNF)

* **2. Visual do Sistema**
  * 🎨 2.1. Desenhar as telas no Figma
  * 🚨 2.2. Definir o estilo visual do Alerta Vermelho de estoque

* **3. Banco de Dados**
  * 📐 3.1. Desenhar a modelagem do banco (DER) no draw.io
  * 💾 3.2. Configurar o banco SQL Server Express
  * 🚀 3.3. Criar as tabelas automáticas usando Migrations

* **4. Código em C# / .NET**
  * 📦 4.1. Criar as classes de Produto, Grade e Fornecedor (`SuaLoja.Model`)
  * ⚙️ 4.2. Programar a lógica que detecta a falta de estoque na fábrica
  * 🖥️ 4.3. Construir o painel do administrador e as telas do estoque

* **5. Testes e Entrega**
  * 🧪 5.1. Testar o sistema com dados reais de calçados
  * 📦 5.2. Subir o código final completo no GitHub
