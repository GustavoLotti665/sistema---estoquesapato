Este documento detalha o plano de ação técnico e a execução prática para o desenvolvimento do sistema.

### 🛠️ Planejamento de Ação (5W)

| Diretriz | Descrição do Plano de Ação |
| :--- | :--- |
| **❓ What?** | **Desenvolver** um software de gestão de estoque e pedidos para e-commerce e **integrar** a plataforma com os dados de disponibilidade dos fornecedores. |
| **🎯 Why?** | **Eliminar** os furos de estoque causados pela venda de calçados indisponíveis e **reduzir** o índice de cancelamento de pedidos. |
| **📍 Where?** | No ambiente residencial e no local de trabalho do e-commerce. |
| **⏰ When?** | Durante o primeiro semestre letivo de 2026, seguindo o calendário das disciplinas. |
| **👤 Who?** | Gustavo Lotti, com o apoio do gerente da loja de calçados para a validação das regras de negócio. |

| Diretriz | Descrição da Estratégia e Recursos |
| :--- | :--- |
