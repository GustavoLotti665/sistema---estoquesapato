# 📜 Artefato 8: Termo de Abertura do Projeto (TAP)

Este documento formaliza o início do projeto, definindo seus objetivos principais, as pessoas envolvidas e as restrições de alto nível.

---

### 📌 1. Informações Básicas
* **Nome do Projeto:** Sistema Integrado de Gestão de Estoque (HJM Calçado)
* **Gerente do Projeto:** Gustavo Lotti
* **Data de Início:** Março de 2026
* **Previsão de Conclusão:** Fim do 1º semestre letivo de 2026

---

### 🎯 2. Justificativa e Objetivos
* **O Problema:** A loja sofre com cancelamentos e estornos frequentes porque vende sapatos no e-commerce que já esgotaram na fábrica/fornecedor.
* **O Objetivo:** Desenvolver um software que cruze os pedidos do site com a planilha de disponibilidade do fornecedor, gerando alertas automáticos antes do faturamento, para zerar as falhas de entrega.

---

### 🗺️ 3. Escopo Macro (O que faremos)
* Levantamento de requisitos e criação de documentação técnica (Matrizes, BPMN, UML).
* Desenvolvimento de um backend em C# / .NET Core.
* Criação de um banco de dados relacional (SQL Server) para armazenar os dados de produtos e grades.
* Construção de uma interface de painel (Clean UI) para os administradores da loja.

---

### ⚠️ 4. Premissas e Restrições
* **Premissas:** O fornecedor deve continuar enviando as atualizações de estoque (mesmo que via planilha) para alimentar o sistema.
* **Restrições:** O sistema deve ter custo financeiro zero para a loja, utilizando apenas tecnologias e ferramentas gratuitas (VS Code, draw.io, Figma, SQL Server Express). O prazo de entrega é estritamente limitado ao calendário acadêmico da Fatec.
