# 📦 Sistema Integrado de Gestão de Estoque (HJM Calçados)

> **Projeto acadêmico focado na validação de estoque e controle de fluxo de pedidos para e-commerce.**

## 🎯 Sobre o Projeto
Este projeto foi desenvolvido para resolver um problema crítico e frequente no varejo online: a venda de produtos (como grades específicas de calçados) que já esgotaram no fornecedor/fábrica. 

O sistema cruza automaticamente os pedidos recém-pagos no site com a planilha de disponibilidade do fornecedor. Caso seja detectada a falta da numeração, o sistema bloqueia o faturamento e aciona um alerta para intervenção humana, evitando transtornos com o cliente e custos com estornos.
